## 🚀 部署步骤

### 快速部署（3步）

1. **准备 GitHub 仓库**
   - 创建公开仓库
   - 获取仓库地址

2. **上传代码**
   - 使用 Git 命令行或 GitHub Desktop
   - 将所有文件推送到 GitHub

3. **部署到 Streamlit Cloud**
   - 访问 https://streamlit.io/cloud
   - 连接 GitHub 仓库
   - 设置 Main file path: `app.py`
   - 点击 "Deploy!"
